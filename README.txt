GU-Q Hotel & Serviced Apartment Catalogue — High-Contrast Text Update

What changed
1. Every grey text state across the catalogue now uses high-contrast charcoal or near-black.
   This includes labels, helper text, property location text, badges, amenity chips, rate notes,
   modal details, compare-drawer text, removal icons, footer text, and placeholders.
2. Borders and scrollbars are darker and easier to see without turning the interface heavy.
3. The hero and modal use darker charcoal in their gradient instead of light grey.
4. The prominent rates-and-availability notice and Property Finder footer link from the previous
   readability update are retained.

Files edited
- website/styles.css

Files retained without functional changes
- website/index.html
- website/app.js
- website/config.js
- website/sample-data.js
- apps-script/Code.gs

How to deploy
Replace the contents of your GitHub Pages website folder with the contents of the website folder.
No Apps Script or Google Sheet changes are required.

Preview
Open preview/GUQ_hotel_catalogue_high_contrast_preview.html in a browser. It is self-contained and
uses sample data only, so it does not update or alter your live Google Sheet.
